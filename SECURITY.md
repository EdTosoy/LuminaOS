# Security Policy

## Supported Versions

Currently, only the `main` branch is supported for security updates as we are in the pre-alpha/strategic phase.

| Version | Supported |
| ------- | --------- |
| Alpha   | Yes       |

## Reporting a Vulnerability

We take the security of Lumina OS seriously. If you find a vulnerability, please do not open a public issue. Instead, follow these steps:

1. **Email us**: Send a detailed report to [security@luminaos.org](mailto:security@luminaos.org) (placeholder).
2. **Detail the issue**: Include steps to reproduce, the impact, and any potential fixes.
3. **Wait for response**: We will acknowledge your report within 48 hours and provide a timeline for a fix.

We follow a 90-day disclosure policy. Please allow us time to resolve the issue before making it public.
